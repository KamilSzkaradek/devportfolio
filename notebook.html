Notebook

1.After turning on the program a window with active number buttons from 0 to 9 and a Reset button will appear.





2.Pressing either of the number buttons will make the numbers assigned to them appear in the text box above as one number. That action will also make the rest of the buttons active.





3.Choosing the comma button will add a comma to our number.





4.Doing that will force us to input at least one other number after the decimal point.





5.After our first number is done we can choose either of the operation buttons (add, substract, divide, multiply or equal sign).





6.And input any other number the same way as before.





7.This program limits us to only two numbers, so now only the equal sign is avaliable. After pressing the Reset button program will return to it's original status.
